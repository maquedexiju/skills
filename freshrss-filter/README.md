# FreshRSS 过滤 Skill - 使用说明

## 工作流程

```
┌────────────────────────────────────────────────────────────┐
│ Step 1: 用户调用 skill                                     │
│         "过滤我的 RSS"                                      │
└────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────┐
│ Step 2: Claude 执行 fetch                                  │
│         python freshrss_tool.py fetch                      │
│         - 获取所有未读条目                                  │
│         - Feed 黑名单过滤 (TTG/TCCF/M-Team)                │
│         - 内容正则黑名单过滤                                │
│         - 输出候选条目 JSON                                 │
└────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────┐
│ Step 3: Claude 分析并判断                                  │
│         - 读取 prompt.md 过滤标准                          │
│         - 分析条目内容                                      │
│         - 返回需过滤的 ID 列表                              │
└────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────┐
│ Step 4: Claude 执行 mark                                   │
│         python freshrss_tool.py mark --ids id1,id2,id3     │
│         - 标记指定条目为已读                                │
└────────────────────────────────────────────────────────────┘
```

## 本地过滤规则

所有黑名单文件都支持**正则表达式**，每行一个规则，以 `#` 开头的是注释。

### Feed 黑名单（feed_blacklist.txt）

自动过滤特定 RSS 订阅源：
```regex
# PT 站
(?i)ttg
(?i)totheglory
(?i)tccf
(?i)torrentccf
(?i)m-team
(?i)mt
(?i)tp torrents

# 其他不想看的来源
(?i)某广告源
```

### 作者黑名单（author_blacklist.txt）

过滤特定作者：
```regex
# 广告账号
(?i)spam.*bot
(?i)推广.*账号

# 特定用户
(?i)某用户名
```

### 内容黑名单（content_blacklist.txt）

过滤包含特定关键词的条目：
```regex
# 广告关键词
(?i)(广告|推广|赞助)

# 连续剧（TTG 的多集更新）
(?i)第.*集.*2160p

# 特定类型
(?i)促销|优惠|限时抢购
```

## 日志文件

每次执行会生成带时间戳的日志文件：
```
logs/filter_20260413_204812.log
```

日志包含：
- 执行时间
- 过滤详情
- 统计信息（总条目、各类过滤数量、剩余未读）

## AI 过滤标准（prompt.md）

针对 V2EX 等内容，过滤以下类型：

**过滤（标记已读）：**
- 个人情感故事、婚姻、生育话题
- 闲聊、水贴、无信息量的求推荐
- 争议性话题、站队争论
- 游记、旅游分享
- 广告推广

**保留：**
- 职场经验、求职、工作相关
- 技术问题、编程、工具使用
- 软件/开源项目推荐
- 实用信息、政策解读
- 招聘信息

## 命令参考

### 1. 获取条目（供判断）
```bash
python freshrss_tool.py fetch
```

输出 JSON 格式：
```json
{
  "stats": {
    "total": 214,
    "feed_filtered": 54,
    "content_filtered": 0,
    "remaining": 160
  },
  "feed_filtered": [...],  // 已自动过滤
  "content_filtered": [...], // 已自动过滤
  "candidates": {          // 需 AI 判断
    "V2EX": [...],
    "太妃的市场朗读": [...]
  }
}
```

### 2. 标记已读
```bash
python freshrss_tool.py mark --ids "tag:google.com,...,tag:google.com,..."
```

### 3. 统计
```bash
python freshrss_tool.py stats
```
